# AI Learning Report Example

This example demonstrates a small group classroom report about responsible AI use in coursework.

- Validated Slide Spec: `ai-learning-report.yaml`
- Expected output prefix: `ai-learning-report`
- Expected deliverables:
  - `outputs/ai-learning-report-presentation.pptx`
  - `outputs/ai-learning-report-speaker-notes.md`
  - `outputs/ai-learning-report-preview.png`

Validate it from the plugin package root:

```powershell
python scripts/validate_slide_spec.py examples/ai-learning-report.yaml --json
```
