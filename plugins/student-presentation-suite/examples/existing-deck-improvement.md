# 已有 Deck 改进示例

- 请求：“直接把这个答辩 PPT 改好。”
- 流程：先输出 review diagnosis，再在同一任务进入 `student-presentation-ppt`。
- 交接：使用 `source_deck`、`edit_intent`、`review_findings`、`preserve` 和 `change_summary_required`。
- 门槛：已有 deck 和 review 结论可预填需求表，但仍需补齐缺失字段并确认 Production Summary 后才能修改文件。
- 保留：模板、logo、已批准内容和来源。
- 交付：独立改进版 PPTX、讲稿、预览和 change summary；不得覆盖原文件。
